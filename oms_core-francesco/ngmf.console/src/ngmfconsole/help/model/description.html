model()

model() - defines the model in a simulation.

Synopsis

model() ::= 
    Sequence of:
        Zero or more of:
            parameter()

Description

The model specifies the component that runs at a core of a simulation. Models can be designed in more than one way, therefore the model definition may vary.

Attributes

Name Description Type Required
classname The java component to run as a model String No
iter the method and data to be iterated String No
while the method and data to iterate while a condition is true. String No
until the method and data to iterate until a condition is true. String No
if the method and data to run if a condition is true String No

Parents

These elements contain model(): sim(), luca(), esp(), tests()

Children

The following elements occur in model(): parameter(), components(), connect(), feedback(), logging()

See Also

Examples

model(classname:"wbm.WbmLakeIsoSum") {
   parameter (file:"$oms_prj/data/bluenile1/param.csv") {    
       inputFileDir "$oms_prj/data/bluenile1/"
       pptFileName  "pre.csv"

       sumFile      "basinsum.csv"
       startTime    "1960-01-01"
       endTime      "1990-12-01"
    }
}
